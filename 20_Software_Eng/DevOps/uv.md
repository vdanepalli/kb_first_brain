# UV - Python

**Direct dependencies** are the top-level packages your project needs. 
**Transitive dependencies** are the packages of your direct dependencies. 
**Development dependencies** are packages you only need while developing the project.

**Dependency** is anything external to your own code that is required to get your code to run as expected

Packages, System Software, Operating System, Hardware, Environment

Reproducibility, Deterministic. 

Development | Build | Deploy

**Definition file** is a file where you state the dependencies you need and the minimal version constraints.
